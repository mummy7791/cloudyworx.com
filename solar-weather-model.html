Why it matters

Geomagnetic disturbances (GMDs) from solar storms can disrupt power grids, satellites, aviation, and communications. Accurate early forecasts give operators time to reroute loads, place satellites in safe modes, and protect equipment.

  • Short-term (6–24h) flare & CME risk nowcasting
  • Mid-term (24–72h) Kp, Dst, AE index forecasting
  • Uncertainty-aware outputs for risk-based decisions

What’s inside

  • Architecture: Multimodal foundation model (temporal transformer + CNN encoder) for solar imagery (SDO/AIA), heliospheric parameters, and ground magnetometer series.
  • Inputs: EUV images, solar wind speed/density/Bz, F10.7, OMNI data, and magnetometer arrays.
  • Targets: Probabilistic forecasts for Kp/Dst, CME arrival windows, and storm severity tiers.
  • Outputs: Point forecasts + calibrated prediction intervals (P10/P50/P90).

Quick start

# 1) Install
pip install cloudyworx-solar

# 2) Download weights (or use the “Get the model” button)
python -m cloudyworx_solar.download --model solar-foundation-v1

# 3) Inference (Python)
from cloudyworx_solar import SolarForecaster, load_example_batch

model = SolarForecaster.from_pretrained("solar-foundation-v1")
batch = load_example_batch()  # loads sample AIA images + time-series
pred = model.predict(batch, horizon_hours=48)

print(pred.kp.mean, pred.kp.interval)          # Point & interval forecasts
print(pred.cme.arrival_window)                 # Estimated CME arrival
pred.save_plot("forecast.png")                 # Convenience plotting

Note: Replace package/command names with your actual repo details when ready.
